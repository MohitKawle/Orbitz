
# Clone of Orbitz.com

This project is a clone of orbitz.com, which is a is a travel fare aggregator website. This is collaborative project, completed within 6 days with a team of 6 members. This website have end to end functionalities from SignUp/LogIn and searching the hotel rooms to booking and payment.

## Features

* SignUp/LogIn

* Google SignUp

* Hotel search (based on the city)

* Sorting and Filtering

* Map

* Room Booking

* Payment (through stripe gateway)
